Game Type 1: Train questions from one category, starting from basic towards more advanced questions.
You start with a total of 10 lives. After each mistake that you make, you will lose 1 life and repeat the same question.
You get an extra bonus point (and skip a question) each time you answer 2 consecutive questions correctly.
Each username's highest score will be documented per category so that next game, you don't need to start all over.
In that case, you will start 15 questions before where you left off in your previous best attempt.





Game Type 2: 10 questions of all domains, in all difficulty levels, randomly selected
You will be asked 10 questions, each randomly selected. After each round you will continue only with the questions that were answered incorrectly in the previous round(s). The game will finish after all 10 questions have been answered correctly. However, after each round, the reward (points) for a correctly answered question will be reduced by a factor of 75%. The maximum score is 100 points, which can only be achieved by answering all 10 questions correctly in the first round.
(Note: this set of questions includes also other question categories, including neuroscience and machine learning. Chinese vocabulary training questions are excluded from this game type.)
